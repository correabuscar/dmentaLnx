Things you need to do manually before using this program:
configuration for ubuntu:

to get php5:
* sudo apt-get install php5

postgresql version (non pdo) [currently!]:
* sudo apt-get install php5-pgsql

* sudo apt-get install postgresql-8.2 postgresql-server-dev-8.2 postgresql-dev

* vim /etc/php5/cli/php.ini append "extension=pdo_pgsql.so" at eof

* append the following to: sudo vim /etc/postgresql/8.2/main/pg_hba.conf
local   all         demlinks_user                     password demlinks_db

* sudo /etc/init.d/postgresql-8.2 restart

* ./initdb.sh
        notice that this uses sudo -u postgres  to switch to that user thus for that to work seaminglessly I have the following in /etc/sudoers:
        myunixuser ALL=(ALL) NOPASSWD:ALL
